Ввод данных от пользователя

В Java есть встроенный класс Scanner считываюший ввод польхователя  
Для чтения пользовательних данных необходимо создать экземпляр классы Scanner  
Scanner input = new Scanner(System.in);   
Название класса Scanner совпадает с типом класса Scanner  

Для работы с классом нужен импорт встроенного пакета  
import java.util.Scanner;  

Методы класса Scanner  
nexLine - считыевает строку  
nextInt - считывает целове число  
nextDouble - считывает число с точкой
