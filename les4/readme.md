Как называть переменные

camelCase ротация - первая маленькая, каждое след слово с большо, cлитно.  
Так именуются переменные, методы и поля экземпляра

PascalCase ротация - первая буква большая  
Так именуются классы и интерфейсы

DB_PASSWORD - тип наименования констант
Константы - не меняются во время работы. Назначаются до запуска программы.
Для обозначения константы ставится в начале ключ слово final
final int housNumber = 45;
 
Не использовать непонятные имена переменных: foo, bar, text, xyz  
Использовать понятные наименования, описывающие значение: fullName, studentId

Имена переменных чувствительны к регистру  
Должны начинаться с буквы, знака $ или нижнего подчеркивания _  
Последующие символы: буквы, цифры, $ или _  
Имя не может содержать пробелы  
Ключевые слова Java не могут быть в качестве имени переменной

Рекомендации:  
- Не использовать $ в имени переменной  
- Избегать использование _ в начале имени  
- Имена должны описывать данные (что содержит)
- Использовать имена в нижнем регистре
- Использовать camelCase ротацию


Литерал - это фиксированое значение, которое используется прямо в коде
int age = 20; "20" - Целочисленный литерал
String name = "Dima"; "Dima" - Строчный литерал
