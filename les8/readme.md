Строка

String - является предопреденнным классом в Java  
Каждая строка это объект.   
Каждая переменная класса строка являются ссылочным типом  
Помещаются между двойными ковычками  
Возможно конкатенация через знак + (соединание)  

Строки являются неизмененяемые - строка не можеть быть изменена после создания.  
Любая операция изменяющая строку, на самом деле создает новую.  


Методы строк  
length - получить длину строки  
isBlank - пустая строка или нет   
toUpperCase - Строка в верхнем регистре  
toLowerCase - строка в нижнем регистре  
strip - убирает пустые символы в начале и конце строки  
equals - сравнение двух строк.  

strip убирает пробельные символы: Пробел " ", Табуляция "\t", Новая строка "\n", Возврат каретки "\r", Форматная табуляция "\f"


Спецсимволы в строках  
Спецсимвол - escape последовательность  
\t - табуляция    
\n - переход на новую строку    
_\" - вставляет символ двойной ковычки  
'\\' - вставляет \ в сроку
