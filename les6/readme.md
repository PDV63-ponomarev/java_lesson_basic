Выражения

Выражение - это конструкии, которые используются для получения одного значения.  
Может состоять из переменных, оператоворов или вызовов методов  
Несколько выражений можно объединить в одно, чтобы получить более крупное.   
Одно выражение - одно значение  

5 + 3 = 8  
a > b = true или false  
"Hello " + "World" = "Hello World"  
divide(10, 5) = результат метода (функции)  



Инструкции (statments)

инструкции выполняют определеные действия  
Это единица кода, которая описывает действия, выполняемое программой  
Может быть одной строкой кода или блоком, содержащим несколько строк  
Выполняются последовательно


Выражения, завершенные точки с запятой, становятся выражениями-инструкциями  
выражение-инструкции:  
num = 123; - присвоение значение  
num++; - увеличение значения  
System.out.println("hello"); - вызов метода


Декларирующие инструкции:  
double temperature; - обьявление переменной  
double temperature = 36.6;
