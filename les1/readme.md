public - открытый модификатор доступа.
Контроль доступа классу или методу другими частями программ

void - метод ничего не возвращает
Тип возвращаемого значения

параметр (String[] args) - массив строк
массив аргументов, каждый аргумент является строкой,
передаваемый в java из командной строки

static метод можно вызвать через класс
HelloWorld.main()

при выполнении программы, выполняет метод с именем main
