### Блок кода  
Блок - это код заключенный в фигурные строки {} 
Блок может содержать 0 или более строк кода  
Блок без кода называется пустым  
Код внутри блока рассматрывается как единое целое  
{  
код внутри блока  
}

Услов (statment) if  
Инструкия if выполняет код в зависимости от условия  
Код в блоке if выполняется только если оно истино  
В if можно указывать несколько услвий  
Код if может состоять из несколько блоков  

if (условие){  
код если условие true  
}
