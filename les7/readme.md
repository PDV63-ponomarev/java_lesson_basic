Операторы - это символы, используемые для выполнения определенных действий

Операторы действуют на один или несколько операндов, чтобы получить результат
Операндами могут быть переменные, значения или выражения

а = 10  
а, 10 - операнды  
= - оператор

категории операторов  
унарный - один операнд  
бинарный - два операнда  
тернарный - три операнда

типы операторов:
- арифметические
- логические
- присвоение
- сравнение
- условные

Арифметические операторы: сложение (+), вычитание (-), умножение (*), деление(/),
остаток от деления(%), увеличение(++), уменьшение(--)


Операторы сравнения, возврашают true или false.  
"==" - оператор равенства.   
"!=" - оператор не равенства. 
'<' - меньше  
">" - больше  
"<=" - меньше или равно  
">=" - больше или равно  


Логические операторы работают с выражениями или значениями типа boolean  
"&&" -  И  
"||" - ИЛИ  
"!" - НЕ. Префиксный с 1 операндом  

Выражение1 && Выражение2 - возврашает true если оба выражения яляются true
Выражение1 || Выражение2 - возврашает true если одно из выражений true
!Выражение - Инвертирует значение boolean

операторы И ИЛИ являются операторами короткого замыкания. 
Могут не рассматрываться все выражения, если выражение1 уже дает результат.  
Выражение1 && Выражение2 - если Выражение1 false, то результат сразу false и Выражение2 игнорируется  
Выражение1 || Выражение2 - если вырежние 1 true, то результат сразу true и выражени2 игнорируется

Цепочки операторов
a && b && c && d - true если все значения true
a || b || c || d - true если одно из выражений true
a || b && c || d - рекомендуется ставить скобки для большего понимания

логические операторы и операторы сравнения чаше всего используются в условиях if


Операторы присвоения  
"=" - присвоение значение переменной  
"+=" - добавить и присвоить
"-=" - вычесть и присвоить
"*=" - умножить и присвоить
"/=" - разделить и присвоить
"%=" - остаток от деления и присвоить
